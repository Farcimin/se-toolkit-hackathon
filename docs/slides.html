Context

End-user
Russian-speaking beginners and those interested in Judaism who want to learn how to correctly read and pronounce Hebrew prayers.
Problem
Hebrew pronunciation is hard for beginners, and there is no simple self-study tool that pairs the original text with Cyrillic transliteration, spoken audio, and the actual prayer times for where the user lives.
Product idea
"An audio guide for learning Jewish prayers — Hebrew text, Cyrillic transliteration, Russian translation, pronunciation audio, and daily prayer times."
2 / 5

Implementation

Version 1 — Core

  • Catalog of 7 core prayers in 3 categories
  • Hebrew text + Cyrillic transliteration + Russian translation
  • Audio pronunciation (Hebrew neural TTS)
  • Category filter, clean mobile-first UI
  • FastAPI + PostgreSQL + React, all dockerized
Stack: FastAPI · PostgreSQL 16 · React 18 · nginx · Docker Compose

Version 2 — Polish + Deploy

  • Daily prayer times (Shacharit, Mincha, Maariv, sunrise, sunset) for 10 cities incl. Innopolis
  • Zmanim computed locally via KosherJava algorithm — fully offline
  • User progress tracking with persistent "learned" state
  • Prayer icons, progress bar, tabbed navigation
  • Deployed to Innopolis University VM
TA feedback addressed: no specific feedback was given after V1; V2 followed the originally planned roadmap in full.
